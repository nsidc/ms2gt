MS2GT Tutorials

The best way to learn how to use MS2GT is to work through some examples.

In each of the following tutorials, you will need to order some MODIS data using a tool such as the Earth Observing System (EOS) Data Gateway (EDG). Detailed instructions for using the EDG are not provided in the tutorials found here; however a tutorial for using the EDG is available online.

The MS2GT software uses the mapx library to define grids for various map projection. This library reads two kinds of text files called gpd and mpp files that define a grid and a map projection, respectively. Pre-existing gpd and mpp files referred to in the tutorials can be found in the ms2gt/grids directory.

Each tutorial has a corresponding directory under ms2gt. These directories contain the text files (including any gpd and mpp files) created in in the particular tutorial to save you the trouble of typing them in.

  • Tutorial 1: Gridding 1 km Level 1b Data over Greenland Using mod02.pl
  • Tutorial 2: Gridding 250 m Level 1b Data over Greenland Using mod02.pl
  • Tutorial 3: Gridding 500 m Snow Cover Data over Colorado US Using mod10_l2.pl
  • Tutorial 4: Gridding 1 km Sea Ice Data over the Ross Sea Using mod29.pl
